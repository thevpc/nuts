Version 0.8.9 shaping up

Version 0.8.9 is a major milestone. As part of our commitment to quality, odd-numbered releases (5, 7, 9) are our most stable and polished versions. This release brings better security, TSON v2.0, parallel dependency resolution, and an expanded ecosystem.

Security

  • ADDNew AES/GCM cipher implementation replacing broken DES/CBC
  • ADDNSecureToken: better support for external vaults
  • ADDNSecureString: destroyable secrets support

TSON v2.0

  • ADDNuts becomes the reference implementation of TSON
  • ADDRound-trip parser preserving comments and spaces (CST)
  • ADDNElementPath, NElementNavigator for better navigation
  • REMRemoved regex support (conflicts with expressions)

Performance

  • ADDParallel dependency resolution
  • ADDFaster SSH connections with pooling
  • ADDExec classpath caching — no rebuild on each run

Ecosystem

  • ADDJDK/JRE download support: nuts settings add java --download --version=25
  • ADDNative syntax highlighting for TSON and NTF in IntelliJ, VS Code, Vim, Kate, GEdit, jEdit, Notepad++
  • ADDNew Gradle Dependency Solver
  • ADDSupport for Tomcat, PostgreSQL, Netbeans, Docker/Kubernetes

Version 0.8.8.0 released

nuts v0.8.8 is out. A stable, polished release with better NBulkheadCall concurrency support, NDuration parsing improvements, and various fixes.

  • ADDNBulkheadCall to handle calls limited by concurrent count
  • ADDNDuration.parse(String) to parse user durations
  • FIXNMemorySize issue (throws exception on creation)
  • CHGAdded --sandbox --yes by default to spring apps

Version 0.8.7.0 released — Enterprise Resiliency

nuts v0.8.7 focuses on enterprise usability with a full suite of resiliency tools and a revamped documentation.

  • ADDNSagaCall — Saga design pattern with loops and conditions
  • ADDNCircuitBreakerCall — Circuit breaker pattern
  • ADDNRetryCall — Retry pattern
  • ADDNScoredCallable — Score-based call strategy
  • CHGPrefer NDuration over Duration throughout the API
  • CHGRemoved dependency on AWT, replaced with NColor
  • WARNLicense changed: APACHE2 → LGPL3
2025-01-03 2 min read STABLE
Version 0.8.5.0 released

A major refactoring: nuts is now split into nuts-boot, nuts (api), nuts-lib, and nuts-app. Documentation generation is now handled by the new ndoc/nsite tools.

  • ADDNTemplate: simple text templating with NExpr expressions
  • ADDndoc and nsite-lib for static HTML website generation
  • ADDLevenshtein-based typo detection for CLI arguments
  • ADDNPath::writeObject, writeMsg, writeText
  • CHGnuts split into nuts-boot / nuts / nuts-lib / nuts-runtime
  • WARNLicense changed: APACHE2 → LGPL3

Version 0.8.4.0 — Native Images & Cleaner API

It's been a while. What's new? Cleaner API, native image support, faster startup, better isolation, desktop environment improvements, and much more.

  • ADDNative installers for Windows, Linux and macOS
  • ADDNOptional — feature-rich replacement for java.util.Optional
  • ADDNChronometer, NDuration
  • ADDNProgressMonitor and monitoring API
  • ADDNMemorySize API
  • CHGPrefix changed: NutsX → NX (e.g. NutsPath → NPath)
  • WARNAPI incompatibilities with previous versions
