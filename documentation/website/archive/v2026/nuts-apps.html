Apps & Tools

Java applications you can install and run with a single nuts install command.

🖥 Commandline Tools

🌰
nuts
Package ManagerCLI

The Nuts Package Manager itself. Discover, install, update, and run Java applications from any Maven repository. Multi-version, offline-capable.

$curl -sL https://thevpc.net/nuts/nuts-stable.jar -o nuts.jar && java -jar nuts.jar -Zy

🐚
nsh
ShellCLI

A bash-compatible shell rewritten in Java. Runs on any platform without a POSIX environment. Outputs JSON/XML natively for scripting.

$nuts install nsh
🌐
nsite
Static Site GeneratorCLI

Template-driven static site generator used to build the Nuts website. Processes markdown files with a variable-substitution engine.

$nuts install nsite
📄
ndoc
DocumentationCLI

LaTeX-like documentation tool and folder templater. Replaces variables in source files and generates formatted output.

$nuts install ntexup
📋
njob
Task ManagerCLI

A productivity CLI for tracking tasks, jobs, and projects. Aggregate jobs into projects, tasks into jobs, with status and priority tracking.

$nuts install njob
🔍
ndiff
UtilityCLI

Diff tool for folders, files, and JAR archives. Detect changes between versions of packaged Java artifacts.

$nuts install ndiff
🔎
ncode
Code SearchCLI

Java class and file finder across folders and ZIP/JAR archives. Case-sensitive or insensitive, supports regex patterns.

$nuts install ncode

ndb
DatabaseCLI

Database CLI for Derby, MySQL, and PostgreSQL. Start, stop, and query databases directly from the terminal.

$nuts install ndb
📡
noapi
API DocsCLI

OpenAPI documentation generator. Turn your YAML spec into formatted PDF or HTML documentation.

$nuts install noapi
🔢
nversion
UtilityCLI

File, folder, and project version detector. Extracts version information from Maven projects, DLLs, and more.

$nuts install nversion

nmvn
Build ToolCLI

Maven build tool wrapper managed by Nuts. Run Maven without a local install, with Nuts handling the runtime.

$nuts install nmvn
🌿
spring-cli
Framework ToolCLI

Spring Boot CLI via Nuts. Scaffold projects, run Groovy scripts, and work with the Spring ecosystem from the command line.

$nuts install org.springframework.boot:spring-boot-cli

tsunami
Security ScannerCLI

Google Tsunami: a general-purpose network security scanner with an extensible plugin system for detecting high-severity vulnerabilities.
