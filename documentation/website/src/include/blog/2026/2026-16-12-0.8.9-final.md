# 🚀 Nuts v0.8.9.0 is officially released!

We are excited to announce the release of **Nuts v0.8.9.0**! This version is now available across multiple channels to suit your workflow.

---

## 📦 1. Shell Installer
```
curl -s https://thevpc.net/nuts/install-stable.sh | bash
```

## 📦 2. Maven Central
You can now pull the latest version directly into your Java projects:

```xml
<dependency>
    <groupId>net.thevpc.nuts</groupId>
    <artifactId>nuts</artifactId>
    <version>0.8.9</version>
</dependency>
<dependency>
    <groupId>net.thevpc.nuts</groupId>
    <artifactId>nuts-impl</artifactId>
    <version>0.8.9.0</version>
</dependency>
```

## 💻 3. GitHub Release
The source code and release assets are available on our GitHub Releases page:
👉 [View GitHub Release](https://github.com/thevpc/nuts/releases)
