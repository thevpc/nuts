You can click on various parts of the status bar:
  • Double-clicking the caret position display opens the Go to Line dialog box.
  • Double-clicking the edit mode, folding mode, or encoding indicator opens the Buffer Options dialog box.
  • Clicking the various status flags changes them.
  • Double-clicking the memory indicator opens the Memory Status dialog box.
To configure what components are shown in the status bar, go to the Status Bar pane of the Utilities>Global Options dialog box.
