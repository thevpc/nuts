The action bar can provide quick keyboard access to editor features which do not have explicit keyboard shortcuts. Examples:
  • C+ENTER l-o (automatically completed to global-options)
  • C+ENTER w-v (automatically completed to new-view)
  • C+ENTER buffer.tabSize=4
  • C+ENTER vfs.browser.dock-position=left
  • C+ENTER vfs.browser.shortcut=F3
  • C+ENTER view.fontsize=14
